## Getting Started

#### Installation Process

1. Clone the repo
    ```sh
    https://github.com/codezerro/JdeRobot-gsoc-2025-python-challenge.git
    ```
2. Create python environment
    ```python
    python3 -m venv env
    ```
3. Active the environment
    ```python
    source env/bin/activate
    ```
4. Install package
    ```python
    pip install pygame
    ```
5. Run the app
    ```python
    python3 app.py
    ```
